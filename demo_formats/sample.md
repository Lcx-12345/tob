# 工具演示项目

## 概述

这是一个由 SOLO AI 生成的示例项目，用于演示各种文件格式的生成能力。

## 支持的格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| JSON | .json | 数据交换格式 |
| YAML | .yaml/.yml | 配置文件格式 |
| TOML | .toml | 配置文件格式 |
| XML | .xml | 标记语言 |
| CSV | .csv | 表格数据格式 |
| HTML | .html | 网页标记语言 |
| CSS | .css | 样式表 |
| Python | .py | Python源代码 |
| JavaScript | .js | JavaScript源代码 |
| TypeScript | .ts | TypeScript源代码 |

## 使用方法

```bash
# 运行Python示例
python sample.py

# 运行Shell示例
bash sample.sh

# 运行Go示例
go run sample.go
```
