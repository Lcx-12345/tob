# HelloWorld 程序

这是一个最简单的 HelloWorld 示例。

## Python 版本
```python
print("Hello, World!")
```

## 运行
```bash
python hello.py
```

## JavaScript/Node.js 版本
```javascript
console.log("Hello, World!");
```

## Java 版本
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## C 语言版本
```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## C++ 版本
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## Go 版本
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
